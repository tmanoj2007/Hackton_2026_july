# 🎓 Campus Wallet

A smart AI-powered digital wallet for educational institutions that enables students to make secure cashless payments using QR codes. The system also provides AI-powered spending insights, a Parent Portal, and an Admin Dashboard.

---

## 🚀 Features

- 🔐 Google Authentication (Firebase Auth)
- 💰 Digital Wallet
- 📷 QR Code Payments
- 📜 Transaction History
- 🤖 AI Spending Tips (Gemini AI)
- 👨‍👩‍👧 Parent Portal
- 👨‍💼 Admin Dashboard
- 🔔 Notifications
- ☁️ Firebase Hosting

## ▶️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.git
```

### Install dependencies

```bash
npm install
```

### Configure Firebase

Create a `.env.local` file and add your Firebase and Gemini API keys.

### Run the project

```bash
npm run dev
```
